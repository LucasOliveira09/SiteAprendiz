JavaScript

JavaScript é uma linguagem de programação usada para criar páginas web interativas. Ele é executado no navegador e permite alterar o conteúdo de uma página, responder a eventos e até mesmo interagir com o servidor.

O que é o JavaScript?

JavaScript é uma linguagem de script orientada a objetos e baseada em eventos. Ele é uma das três principais tecnologias da web, ao lado de HTML e CSS.

Exemplo de código em JavaScript

                
    function saudacao() {
        alert('Olá, mundo!');
    }
    saudacao();

Este é um exemplo simples de um código que exibe uma mensagem no navegador ao ser executado.
